# Template Catalog

Read only the template files needed for the current task. Replace every `{placeholder}` with domain-specific, compile-safe code.

| Component | Template file |
| --- | --- |
| Entity | `templates/Entity.template` |
| Action | `templates/Action.template` |
| DTO | `templates/DTO.template` |
| UseCase | `templates/UseCase.template` |
| Infrastructure model | `templates/Model.template` |
| Infrastructure provider | `templates/Provider.template` |
| Repository interface | `templates/RepositoryInterface.template` |
| Repository implementation | `templates/RepositoryImplementation.template` |
| Domain event | `templates/DomainEvent.template` |
| Event handler | `templates/EventHandler.template` |
| Controller | `templates/Controller.template` |
| Request | `templates/Request.template` |
| Rules attribute | `templates/RulesAttribute.template` |
| Base DTO | `templates/BaseDTO.template` |
