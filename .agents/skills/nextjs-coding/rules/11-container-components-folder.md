# Rule 11 - `containers/<url-path-name-container>/components/`

## Purpose
Container-private UI building blocks.

## Must
- Use `.agents/skills/react-component-generator/SKILL.md` when creating/refactoring components.
- Keep components focused on presentation and local interaction.

## Must Not
- Duplicate components already in `components/`.
- Embed route-level orchestration logic.
